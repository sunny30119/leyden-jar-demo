# ⚡ 萊頓瓶人體感應實驗 (Leyden Jar Demo)

一個用 **HTML5 + 原生 Canvas API** 製作的互動式科學教學網頁，透過微觀電荷粒子模擬「萊頓瓶儲能」與「人體靜電感應」的物理原理。

## 🎯 線上體驗

👉 **[點此開啟互動網頁](https://sunny30119.github.io/leyden-jar-demo/)**

## ✨ 功能特色

- **完整人體剪影**：頭、肩膀、雙臂、軀幹、雙腿、心臟標記
- **萊頓瓶橫截面**：內層導體 / 玻璃絕緣 / 外層導體（會隨失去電子變色）
- **可拖曳互動**
  - 黃色靜電棒（帶負電）— 靠近瓶頂 + 按住滑鼠注入電子
  - 右手 — 拖向瓶頂金屬球觸發放電，鬆開後自動回位
- **靜電感應視覺化**：同性相斥算法把電子從左手「推」向四肢
- **閉合迴路放電**：電弧 → 右手 → 軀幹 → 左手 → 中和外層
- **三種教學模式**
  1. 純互動沙盒
  2. 步驟引導
  3. 完整教學（含側邊原理說明）
- **慢動作按鈕**（0.25× 速度）— 讓電子流動看得更清楚
- **即時數據面板**：內層電荷量、人體感應流動量、預估電壓
- **深色主題 + 放電閃爍特效**

## 🛠 技術規範

- 純原生 HTML / CSS / JavaScript
- **不使用任何第三方套件**（無 PixiJS、Three.js）
- 僅使用 Canvas 2D API 繪圖
- 支援桌機與平板（RWD）

## 📖 使用方式

1. 拖曳黃色靜電棒到瓶頂金屬球附近
2. 按住滑鼠左鍵 → 注入電子到內層
3. 觀察人體電子被斥力推向四肢
4. 拖右手靠近瓶頂 → 觸發放電
5. 隨時按「慢動作」看細節，按「重置」歸零
